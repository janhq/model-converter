# Introduction
This repository is a Model Converter tool, designed for converting Huggingface Large Language Models into various specified formats.

# Supported Formats
- [x] GGUF (Q3_K_M, Q4_K,M, Q5_K_M, Q8_0)
- [ ] AWQ
- [ ] GPTQ
- [ ] Tensorrt-LLM
